# 数据目录说明

## 📁 目录结构

```
data/
├── raw/                    # 原始数据
│   └── sample_data.jsonl   # 示例数据(10条)
└── processed/              # 处理后数据
    └── (训练时生成)
```

## 📊 数据格式

### 原始数据格式 (JSONL)
```json
{
  "question": "什么是生产要素？",
  "answer": "生产要素是指用于生产物品和劳务的任何东西，包括土地、劳动力、资本和企业家才能。"
}
```

## 🔄 数据处理流程

1. **文本提取**: 从教材中提取经济学内容
2. **预处理**: 清理和格式化文本
3. **问答生成**: 使用AI生成问答对
4. **质量控制**: 人工审核和修正

## 📝 使用说明

- `sample_data.jsonl`: 包含10条示例数据，用于测试
- 完整数据集包含1026条专业经济学问答对
- 数据基于《经济学原理》(N.格里高利·曼昆)教材生成
